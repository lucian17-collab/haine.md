# 🛍️ FashionHub - Magazin de Haine Online Premium

Un magazin de haine modern și atragator, creat cu HTML5, CSS3 și JavaScript vanilla cu design responsiv și imagini de înaltă calitate.

## ✨ Caracteristici

- **Design Modern și Responsiv** - Funcționează perfect pe telefon, tablă și calculator
- **Imagini Reale** - Toate produsele au imagini de calitate de pe Unsplash
- **12 Produse Premium** - Colecție variată de tricouri, pantaloni, rochii, jachete și mai mult
- **Prețuri în RON** - Prețuri competitive în lei românesc
- **Filtrare după Categorie** - Bărbați, Femei și Copii
- **Coș de Cumpărături Funcțional** - Adaugă, șterge și modifică cantități cu imagini preview
- **Local Storage** - Coșul tău este salvat automat
- **Animații Smooth** - Tranzițiile sunt fluide și plăcute
- **Notificări în Timp Real** - Se afișează mesaje la fiecare acțiune
- **Wishlist** - Marchează produsele favorite

## 📂 Structura Fișierelor

```
magazin de hain/
├── index.html      # Pagina principală
├── styles.css      # Stilurile și design-ul
├── script.js       # Funcționalitățile JavaScript
└── README.md       # Acest fișier
```

## 🚀 Cum se Folosește

1. **Deschide index.html** în browserul tău
2. **Explorează Colecția** - Culege prin produse și filtrează după categorie
3. **Adaugă în Coș** - Apasă butonul "Adaugă în Coș" pentru orice produs
4. **Vizualizează Coșul** - Apasă pictograma 🛒 pentru a vedea coșul tău
5. **Modifică Cantități** - Folosește + și - pentru a ajusta cantitățile
6. **Finalizează Comanda** - Apasă "Finalizează Comanda" pentru checkout
